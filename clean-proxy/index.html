Clean Proxy

A cleaner looking proxy!

You may be asking yourself this: What in the world is Clean Proxy?! Well! It's a plugin to make the default BungeeCord commands simpler to understand and quite simply more stylish. I've also added a few other commands which you can see below. I also plan on adding configurations to make your server more customisable. As this project is in a beta stage, please report any bugs to Buddeh at twitter.com/BuddehDev. Feedback is also welcome. :D

Commands
Command Description Permission Node Other commands Usage
Alert Broadcasts a message to the network! bungeecord.command.alert None /alert (json message)
Alertraw Broadcasts a json message to the network! bungeecord.command.alert None /alert (json message)
Cleanproxy Displays a bunch of useful information! None None /cleanproxy
Find Finds a player's location on the network! bungeecord.command.find None /find (player)
Info Provides you with key information about a player! cleanproxy.command.info None /info (player)
IP This command has no point, literally. bungeecord.command.ip None /ip
Glist Lists all online players! bungeecord.command.list None /glist
Gplugins Lists all the BungeeCord's plugins! cleanproxy.command.plugins /gpl /gplugins
Send Provides options for sending players around the network! bungeecord.command.send None /glist
Server Lists all servers with the server's information. bungeecord.command.server None /server or /server list
